Why we’re building this

IoT means the “Internet of Things” — everyday objects that send data to each other. Your phone, your smart speaker, the temperature sensor in a greenhouse. By the end of this course you’ll understand how that works because you’ll have built one yourself, from the SD card up. You’ll learn:

  • How to set up a small computer (a Raspberry Pi) from scratch.
  • How sensors talk to each other using a protocol called MQTT.
  • How to store data in a database and view it on a screen.
  • How to use an AI like Claude through an API to explain data.
